Thanks for purchasing this Kickoff html5/css3 theme

Kickoff is an HTML template. This template gives simple and elegance look and feel no messy layout with overflow of content. Customization is fun to do with this template, everything is managed and well commented. If you feel any issues please read this documentation and you will get to know everything, if you still need any help or you want me to customize it for you, you may send me an email at support@kodeforest.com.

Understanding Grid System

Back to Top

This template use bootstrap grid to make it very easy for anybody to understand the grid system. There is some basic details about its html structure below.

Parent Classes of Sections

.kode-blog-list class is created parent class over Blog Listing,.kode-grid-blog class is created parent class over Blog Grid,.kode-fullwidth-blog class is created for blog detail.

A little bit step onto innovation

.kode-subheader class we have created this class to manage sub header which is coming under menu bar it take background and its position we have used bootstrap classes for child elements to make it maximum mobile friendly / Responsive.

.subheader-height class we have created to manage sub header height to manage its breadcrumbs and page caption which is coming under the menu.
